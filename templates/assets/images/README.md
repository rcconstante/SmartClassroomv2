# DLSUD Logo Setup

## Instructions

1. Save the DLSUD logo image you provided as `dlsud-logo.png`
2. Place it in the directory: `templates/assets/images/dlsud-logo.png`

The logo will automatically appear in:
- Sidebar (top left)
- Login page (left hero section)

## Image Requirements

- **Format**: PNG (with transparency recommended)
- **Recommended size**: 200x200 pixels or larger
- **File name**: `dlsud-logo.png`

## Fallback

If the logo image is not found, the app will display a default SVG icon as a fallback.
